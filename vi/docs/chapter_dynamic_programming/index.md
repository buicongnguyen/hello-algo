# Quy hoạch động

Quy hoạch động phân rã bài toán thành các bài toán con chồng lặp, lưu kết quả và xây dựng nghiệm lớn từ nghiệm nhỏ.

![Quy hoạch động](../assets/covers/chapter_dynamic_programming.jpg)

Chương này đi từ đệ quy và ghi nhớ đến bảng $dp$, sau đó áp dụng cho đường đi, ba lô, đổi tiền và khoảng cách chỉnh sửa.
