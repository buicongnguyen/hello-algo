# Heap

![Heap](../assets/covers/chapter_heap.jpg)

Heap là cây nhị phân hoàn chỉnh duy trì thứ tự ưu tiên giữa nút cha và nút con. Cấu trúc này hỗ trợ truy cập phần tử ưu tiên cao nhất trong thời gian hằng số và cập nhật trong thời gian logarit.

Chương này trình bày thao tác heap, cách xây heap và bài toán top-k.
