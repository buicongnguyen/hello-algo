# Cùng đóng góp

Dự án hoan nghênh sửa lỗi kỹ thuật, cải thiện diễn đạt, bổ sung ví dụ và phản biện bản dịch. Trước khi thay đổi lớn, hãy đọc hướng dẫn đóng góp và thảo luận phạm vi trong issue.

## Chỉnh sửa nhỏ

Mỗi trang nguồn trên GitHub có thể mở bằng nút chỉnh sửa. Tạo fork, sửa Markdown trên nhánh riêng, kiểm tra liên kết và gửi pull request mô tả rõ lý do thay đổi.

![Nút chỉnh sửa trang](contribution.assets/edit_markdown.png)

## Soạn nội dung

- Giữ thuật ngữ nhất quán với bảng thuật ngữ.
- Ghi rõ nguồn và không xóa thông tin giấy phép.
- Ưu tiên ví dụ Python ngắn, có thể chạy độc lập.
- Kiểm tra hình, công thức, liên kết và bản dựng trước khi gửi.
- Tách thay đổi không liên quan thành pull request khác.
