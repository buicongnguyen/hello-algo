# Cài đặt môi trường lập trình

## Cài đặt IDE

Visual Studio Code là lựa chọn gọn nhẹ, đa nền tảng. Tải từ trang chính thức, cài đặt rồi thêm tiện ích cho ngôn ngữ cần sử dụng.

![Cài tiện ích VS Code](installation.assets/vscode_extension_installation.png)

## Cài đặt môi trường ngôn ngữ

Với Python, cài bản Python 3 đang được hỗ trợ và kiểm tra trong terminal:

```bash
python --version
python path/to/example.py
```

Các ngôn ngữ biên dịch như C/C++, Java, C#, Go, Swift, Rust và Dart cần SDK hoặc trình biên dịch tương ứng. JavaScript và TypeScript thường dùng Node.js. Sau khi cài, kiểm tra lệnh phiên bản và chạy một chương trình nhỏ trước khi mở toàn bộ dự án.

Trên Windows, nên bật tùy chọn thêm công cụ vào `PATH`; nếu lệnh chưa nhận, mở terminal mới sau khi cài đặt.
