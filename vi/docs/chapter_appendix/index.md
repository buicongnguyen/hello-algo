# Phụ lục

Phụ lục tập hợp hướng dẫn thiết lập môi trường lập trình, quy trình đóng góp và bảng thuật ngữ dùng trong bản đọc.

![Phụ lục](../assets/covers/chapter_appendix.jpg)

Các hướng dẫn được viết để người học có thể chạy ví dụ Python ngay, đồng thời biết cách đề xuất sửa lỗi hoặc bổ sung bản dịch trên GitHub.
