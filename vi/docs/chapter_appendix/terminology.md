# Bảng thuật ngữ

Bảng dưới đây chuẩn hóa các thuật ngữ cốt lõi dùng trong bản dịch. Khi một thuật ngữ có nhiều cách dịch, ngữ cảnh kỹ thuật và tính nhất quán trong toàn bộ sách được ưu tiên.

| English | Tiếng Việt |
| --- | --- |
| algorithm | thuật toán |
| data structure | cấu trúc dữ liệu |
| time complexity | độ phức tạp thời gian |
| space complexity | độ phức tạp không gian |
| recursion | đệ quy |
| array | mảng |
| linked list | danh sách liên kết |
| stack | ngăn xếp |
| queue | hàng đợi |
| hash table | bảng băm |
| binary tree | cây nhị phân |
| breadth-first search | tìm kiếm theo chiều rộng |
| depth-first search | tìm kiếm theo chiều sâu |
| binary search | tìm kiếm nhị phân |
| sorting algorithm | thuật toán sắp xếp |
| divide and conquer | chia để trị |
| backtracking | quay lui |
| pruning | cắt tỉa |
| dynamic programming | quy hoạch động |
| greedy algorithm | thuật toán tham lam |
