# Cây

![Cây](../assets/covers/chapter_tree.jpg)

Cây là cấu trúc dữ liệu phi tuyến biểu diễn quan hệ phân cấp. Mỗi nút có thể nối tới các nút con, nhờ đó một tập dữ liệu lớn được chia thành nhiều cây con nhỏ hơn.

Chương này giới thiệu cây nhị phân, các phép duyệt, cách biểu diễn bằng mảng, cây tìm kiếm nhị phân và cây AVL tự cân bằng.
