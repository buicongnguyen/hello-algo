# Hello Algo Atlas

This repository keeps the English Hello Algo source and adds a dependency-light visual companion at the repository root.

The atlas includes:

- a clickable learning-dependency graph;
- an interactive data-structure field guide and operation-cost matrix;
- a step-by-step BFS and DFS traversal lab;
- an interactive complexity chart;
- a step-by-step binary-search interval lab;
- an eight-algorithm sorting comparison;
- a problem-clue decision graph;
- visual comparisons of divide and conquer, backtracking, dynamic programming, and greedy reasoning;
- direct routes into the original English Hello Algo chapters.

Run `npm start` to preview the page locally, or `npm run build` to validate it and create the small `dist` publishing artifact. The included GitHub Pages workflow deploys that artifact when this repository is pushed to a GitHub repository with Pages enabled through GitHub Actions.
