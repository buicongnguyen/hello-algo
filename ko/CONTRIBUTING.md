# 한국어 번역 기여 안내

1. 영어 원문과 잠긴 소스 커밋 `a3166c201853739213d5a3a31b1e4a237aaf1076`을 먼저 확인합니다.
2. [`glossary.md`](glossary.md)와 [`style-guide.md`](style-guide.md)를 따릅니다.
3. 제목, 수식, 링크, 이미지, 코드의 의미와 실행 결과를 보존합니다.
4. 기술 검토와 한국어 문장 검토를 분리해 수행합니다.
5. `npm run check`와 `npm run build`를 실행합니다.

문서는 자체 검토와 빌드 검사를 통과하면 `pilot`, 독립적인 기술·언어 검토까지 끝나면 `published`로 표시합니다. AI가 만든 초안은 독립 검토로 간주하지 않습니다.
