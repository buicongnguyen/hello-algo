# 9장 요약

## 핵심 내용

- 그래프는 정점과 간선으로 구성되며 방향, 가중치, 여러 연결 요소를 가질 수 있습니다.
- 인접 행렬은 간선 조회가 빠르지만 $O(n^2)$ 공간을 사용합니다.
- 인접 리스트는 $O(n + m)$ 공간으로 희소 그래프에 적합합니다.
- BFS는 큐를, DFS는 재귀 또는 스택을 사용합니다.
- `visited` 집합은 사이클에서 정점을 다시 방문하지 않도록 합니다.

## 복습 질문

1. 언제 인접 행렬이 인접 리스트보다 적합할까요?
2. BFS가 무가중 그래프의 최단 경로를 찾는 이유는 무엇일까요?
3. DFS에서 방문 집합을 생략하면 어떤 문제가 생길까요?
