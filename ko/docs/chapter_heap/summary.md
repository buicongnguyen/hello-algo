# 8장 요약

## 핵심 내용

- 힙은 지역적인 순서 조건을 만족하는 완전 이진 트리입니다.
- 배열은 힙을 촘촘하게 저장하고 부모와 자식 인덱스를 즉시 계산합니다.
- `peek`는 $O(1)$, `push`와 `pop`은 $O(\log n)$입니다.
- 아래에서 위로 힙을 구성하면 $O(n)$입니다.
- 크기 $k$인 힙은 top-k를 $O(n \log k)$에 해결하고 스트림도 처리합니다.
