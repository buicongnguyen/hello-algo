Visual Studio Code는 가볍고 여러 운영 체제를 지원합니다. 공식 사이트에서 내려받아 설치하고 사용할 언어의 확장 기능을 추가합니다.

## 언어 환경 설치

Python은 지원 중인 Python 3 버전을 설치하고 터미널에서 확인합니다.

```bash
python --version
python path/to/example.py
```

C/C++, Java, C#, Go, Swift, Rust, Dart 같은 컴파일 언어는 해당 SDK나 컴파일러가 필요합니다. JavaScript와 TypeScript는 일반적으로 Node.js를 사용합니다. 설치 후 버전 명령과 작은 프로그램으로 환경을 먼저 확인합니다.

Windows에서는 도구를 `PATH`에 추가하는 옵션을 선택하고 설치 뒤 새 터미널을 열어야 할 수 있습니다.
