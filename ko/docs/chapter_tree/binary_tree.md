# 이진 트리

이진 트리는 각 노드가 최대 두 개의 자식, 즉 왼쪽 자식과 오른쪽 자식을 갖는 트리입니다. 노드는 일반적으로 값과 두 자식 참조를 저장합니다.

```python
class TreeNode:
    def __init__(self, value: int):
        self.value = value
        self.left = None
        self.right = None
```

![이진 트리의 정의](binary_tree.assets/binary_tree_definition.png)

## 기본 용어

- 루트 노드는 부모가 없습니다.
- 리프 노드는 자식이 없습니다.
- 간선은 인접한 두 노드를 연결합니다.
- 노드의 깊이는 루트에서 해당 노드까지의 간선 수입니다.
- 노드의 높이는 해당 노드에서 리프까지 가장 긴 경로의 간선 수입니다.

![이진 트리 용어](binary_tree.assets/binary_tree_terminology.png)

## 이진 트리의 종류

포화 이진 트리는 모든 레벨이 가득 찹니다. 정 이진 트리는 모든 노드의 자식 수가 0 또는 2입니다. 완전 이진 트리는 위에서 아래로, 마지막 레벨에서는 왼쪽부터 채웁니다. 균형 이진 트리는 두 서브트리의 높이 차이를 작게 유지합니다.
