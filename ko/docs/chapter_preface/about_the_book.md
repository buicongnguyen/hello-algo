# 이 책에 관하여

## 누구를 위한 책인가요?

Hello Algo는 자료구조와 알고리즘을 처음 배우는 사람을 위한 오픈 소스 입문서입니다. 복잡한 수학보다 그림, 애니메이션, 실행 가능한 코드를 통해 핵심 개념을 단계적으로 설명합니다.

![책의 학습 구성](about_the_book.assets/hello_algo_mindmap.png)

이 책은 다음과 같은 독자에게 특히 유용합니다.

- 프로그래밍 기초를 익혔고 문제 해결 능력을 키우고 싶은 학습자
- 알고리즘 면접이나 수업을 준비하며 지식을 체계화하려는 개발자
- 구현뿐 아니라 시간과 공간 비용을 설명하고 싶은 사람

## 무엇을 배우나요?

먼저 복잡도 분석과 기본 자료구조를 배우고, 탐색과 정렬을 거쳐 분할 정복, 백트래킹, 동적 계획법, 그리디 알고리즘으로 확장합니다. 각 장은 개념, 그림, 코드, 연습의 흐름으로 이어집니다.

## 오픈 소스 학습 공동체

책의 글, 그림, 코드에는 많은 기여자의 노력이 담겨 있습니다. 질문과 수정 제안, 번역 검토는 모두 학습 자료를 더 정확하고 친절하게 만듭니다. 이 한국어판도 자체 검토를 통과한 파일럿이며 독립적인 한국어 기술 검토를 기다리고 있습니다.
