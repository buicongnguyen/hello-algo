## 코드를 직접 실행하세요

예제는 읽는 것만으로 끝내지 말고 입력을 바꾸어 실행하세요. 한 줄씩 추적하고, 경계값과 빈 입력을 넣고, 예상 결과를 먼저 적은 다음 실제 결과와 비교합니다.

```python
def sum_to_n(n: int) -> int:
    total = 0
    for value in range(1, n + 1):
        total += value
    return total

print(sum_to_n(5))  # 15
```
