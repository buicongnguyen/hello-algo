# 10장 요약

## 핵심 내용

- 이진 탐색은 매 단계 탐색 공간을 절반으로 줄입니다.
- 일관된 구간 규칙이 경계 오류를 방지합니다.
- 삽입 위치를 이용해 중복 값의 왼쪽과 오른쪽 경계를 구할 수 있습니다.
- 해시 테이블은 흔히 $O(n)$ 공간을 사용해 탐색 시간을 줄입니다.
- 적절한 탐색 방법은 데이터 순서, 조회 횟수, 갱신 빈도에 따라 달라집니다.

## 복습 질문

1. 이진 탐색에 무작위 접근이 필요한 이유는 무엇일까요?
2. 오른쪽 경계는 목표 뒤 삽입 위치와 어떤 관계일까요?
3. 해시 테이블 구축 비용이 가치가 없는 경우는 언제일까요?
