# Hello Algo 한국어판

이 디렉터리는 [Hello Algo](https://github.com/krahets/hello-algo)의 비공식 한국어 번역 초안입니다. 영어 원문 커밋 `4935d2d3877a6205008d89def8d2ba43f7e06275`를 기준으로 하며, 완성된 한국어 Atlas와 0–8장의 문서 초안 48개를 제공합니다. 3–8장에는 한국어 설명, 그림, 수식, 표, Python 예제가 포함됩니다. 원문보다 간략한 문서는 구조와 예제를 보완하고 검토 기준을 통과하기 전까지 파일럿 번역으로 간주하지 않습니다.
